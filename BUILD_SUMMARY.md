## 🏗️ Architecture

```
index.html
    ↓
main.tsx (BrowserRouter)
    ↓
App Component
    ├─ Login Page (if not logged in)
    │   └─ Select role (3 options)
    │       └─ Set userRole
    │
    └─ Dashboard (if logged in)
        ├─ Sidebar (8 filtered menu items)
        ├─ Navbar (profile dropdown)
        └─ AdminOverview (complete dashboard)
            ├─ KPI Cards (4)
            ├─ Charts (2 placeholders)
            ├─ Table (5 rows)
            ├─ Activity (4 items)
            └─ Stats (4 metrics)
```
